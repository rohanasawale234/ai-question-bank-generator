# AI Question Bank Generator

This project generates exam questions from text or PDF documents using NLP and transformer models.

## Features

- Upload syllabus PDF
- Generate meaningful questions using AI
- Automatic question paper generator
- Download question paper as PDF

## Tech Stack

- FastAPI
- spaCy
- HuggingFace Transformers
- Bootstrap

## Installation

pip install -r requirements.txt

## Run

uvicorn app:app --reload
